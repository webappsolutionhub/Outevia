# Outevia - Document Image Viewer 

![Outevia Logo](https://outevia.com/assets/logo.png)

Outevia is a Windows application designed to optimize visual analysis inside documents and email messages. It acts as an intelligent visualization layer you use while reading text documents, PDFs, or handling emails, allowing you to instantly isolate and enlarge any selectable image without changing the document layout or zoom, so you can keep your focus on what you are reading.

---

## Key Features

- **Instant image zoom from documents and emails**  
  Enlarge images, charts, and visual details directly from Word documents, PDFs, or Outlook emails without taking screenshots, saving files, or opening external programs.

- **Non-intrusive viewing layer**  
  Outevia works as an overlay viewer that does not alter the zoom level or layout of the source document, so the reading flow remains unchanged while you inspect visual details.

- **Global shortcut workflow**  
  A global keyboard shortcut (default: `Ctrl + Alt + 0`) triggers the automated workflow: Outevia programmatically selects and copies the graphic element under focus and sends it to a dedicated high‑performance viewer window.

- **Borderless WPF viewer**  
  The viewer is built with WPF and displayed in a borderless, transparency‑enabled window to ensure immediate, distraction‑free consultation of the acquired image.

- **Advanced image navigation tools**  
  Inside the viewer you can:
  - Use mouse‑wheel dynamic zoom  
  - Pan by dragging to explore details  
  - Rotate images  
  - Switch to full‑screen visualization

- **OLE object support**  
  Outevia transparently handles OLE objects generated by Word or Outlook, allowing you to preview embedded graphical content without leaving your usual workflow.

- **.NET‑based, resource‑efficient architecture**  
  The .NET architecture ensures instant responsiveness and low resource usage, making Outevia suitable for intensive professional use.

---

## Use Cases

Outevia is designed for professionals who need to inspect technical diagrams, charts, or photographic attachments with high precision directly from their everyday workflow in Windows. Typical scenarios include:

- Examining technical schematics embedded in Word or PDF documents  
- Inspecting charts and graphs inside business reports or emails  
- Checking photographic details in embedded images without detaching them from the document context  

---

## Quick Start

1. Install Outevia on Windows.  
2. Open a document or email containing an image or graphic.  
3. Place the focus on the desired visual element.  
4. Press the global shortcut `Ctrl + Alt + 0`.  
5. The selected image is copied and opened instantly in the Outevia viewer, ready for zooming and inspection.

## FAQ

### Which versions of Windows are supported?

Outevia supports Windows 10 (version 1903+) and Windows 11. It requires .NET 6 Runtime or later, which can be downloaded for free from Microsoft.

### How do I activate the quick viewer?

Use the predefined key combination `Ctrl + Shift + 0` (or your customized shortcut) in any document or email message. Outevia will open immediately, displaying the selected image content.

### Does Outevia support OLE files?

Yes. Outevia natively handles OLE objects embedded in documents, enabling preview and extraction of embedded content without external applications.

### Does the software require an internet connection?

No. Outevia works completely offline.

### Does Outevia preserve image quality?

Yes. Outevia preserves the original image quality when possible, with a maximum zoom level of 800% to allow optimal exploration of fine details.

### What happens if an image cannot be acquired?

If Outevia cannot directly acquire an image, it automatically activates a screen‑clipping tool so you can capture the desired region and manage it as a standard acquired image.

### Does Outevia provide quick tools?

Yes. Outevia offers quick tools for text annotations, drawing, and OCR to annotate information, provide visual indications, or extract text from the image. These tools can be enabled from the Outevia settings.

### Is Outevia free?

Yes. Outevia is free.
